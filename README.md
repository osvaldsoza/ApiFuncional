<h1>API Funcional</h1>

<h3>Implementação de um CRUD simples para consolidar conhecimentos iniciais adquiridos da paltaforma DotNet Core WebApi.</h3>

## ✔️ Técnicas e tecnologias utilizadas
- ``DotNet 7.0``
- ``Entityframework``
- ``Entityframework Migrations``
- ``Swagger``
- ``Docker``
- ``Docker-compose``
- ``Postgres``
