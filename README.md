<h1>API Funcional</h1>

<h3>Implementação de um CRUD simples para consolidar conhecimentos iniciais adquiridos da paltaforma DotNet Core WebApi.</h3>

<h4>Tecnologias</h4>
<ul>
 <li>DotNet 7.0</li>
 <li>Entityframework</li>
 <li>Entityframework Migrations</li>
 <li>Swagger</li>
 <li>Docker</li>
 <li>Docker-compose</li>
 <li>Postgres</li>
</ul>
